## 戦闘系プラグイン

| リンク | 説明 |
|:---|:--|
| [ダメージ効果音](MSM_AllDamageSE.js) | MPダメージ・HPMP吸収時、HPダメージと同様のSEをつけます。 |
| [ダメージレート](MSM_DamageRateByState.js) | 攻撃対象のステートに応じてダメージ量を増やせます。<br>「毒状態と猛毒状態でダメージ量を変えたい！」という場合に使えます。 |

## マップ系プラグイン

| リンク | 説明 |
|:---|:--|
| [ショップ価格変動](MSM_ShopRate.js) | ショップでの購入・売却価格レートを変数で操作できます。<br>物価上昇をゲーム内で再現可能！ |
| [マップ上で特徴・ステート効果無視](MSM_noStateOnMap.js) | マップ上でHP再生やターン経過を無視できます。 |
